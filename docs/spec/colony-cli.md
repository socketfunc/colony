# colony-cli

gRPCで通信をする

|key|type|example|
|--|--|--|
|namespace|string|`--namespace=default`|

### init

```
$ colony-cli init
```

### describe

詳細情報を取得する

```
$ colony-cli describe
```

### deploy

```
$ colony-cli deploy
```

### list

```
$ colony-cli list
```

### rollback

```
$ colony-cli rollback
```

### delete

```
$ colony-cli delete
```

### version

コマンドのバージョン情報

```
$ colony-cli version
```
